## How to play:

* Compile with:

```bash
make all
```
* Launch the game: _(with the executable and a map.ber)_

```bash
./so_long maps/map1.ber
```

* And play with:
  
  * _Up_ -> `W`
  * _Down_ -> `S`
  * _Left_ -> `A`
  * _Right_ -> `D`
